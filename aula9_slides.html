Limitações do RAG TextualAula 9

1/4

O RAG baseado em chunks de texto não captura relações entre entidades espalhadas pelo corpus.

  • Query de relação: 'Quais crimes o suspeito X cometeu em conjunto com Y?' — requer cruzar múltiplos docs
  • Coreferência: 'ele', 'o réu', 'o acusado' — sem grafo, são entidades disjuntas
  • Análise de rede: vínculos entre organizações criminosas — invisíveis para busca vetorial
  • Solução: Graph RAG extrai entidades e relações, constrói grafo de conhecimento
💡

Arquitetura do LightRAGAula 9

2/4

LightRAG combina grafo de conhecimento com vector store para 4 modos de busca complementares.

  • Extração: LLM extrai entidades (PESSOA, ORGANIZAÇÃO, LEI) e relações do texto
  • Armazenamento: grafo de entidades/relações + vector store de chunks
  • Modo local: busca entidades específicas e seus vizinhos diretos no grafo
  • Modo global: análise de temas e padrões em todo o corpus
  • Modo hybrid: combina local + global para consultas complexas
Python 3.11+
from lightrag import LightRAG, QueryParam
from lightrag.llm import openai_complete_if_cache
from lightrag.utils import EmbeddingFunc

async def llm_func(prompt, **kwargs):
    return await openai_complete_if_cache(
        "meta-llama/Meta-Llama-3.1-8B-Instruct",
        prompt,
        base_url="http://localhost:8000/v1",
        **kwargs
    )

rag = LightRAG(
    working_dir="./juridico_graph",
    llm_model_func=llm_func,
    embedding_func=EmbeddingFunc(
        embedding_dim=1024,
        max_token_size=8192,
        func=bge_m3_embed
    )
)

await rag.ainsert(corpus_juridico_text)

4 Modos de QueryAula 9

3/4

LightRAG suporta 4 modos de busca para diferentes tipos de consulta investigativa.

  • naive: RAG textual clássico — baseline de comparação
  • local: entidade específica + vizinhos no grafo — 'Quem é João Silva?'
  • global: análise temática global do corpus — 'Quais são os padrões de corrupção?'
  • hybrid: combina local + global — melhor para consultas complexas
Python 3.11+
# Modos de query no LightRAG
results = {}

for mode in ["naive", "local", "global", "hybrid"]:
    result = await rag.aquery(
        "Quais são as conexões entre o Cartel X e autoridades públicas?",
        param=QueryParam(mode=mode)
    )
    results[mode] = result
    print(f"[{mode}] {result[:200]}\n")

# hybrid geralmente mais rico para queries investigativas

Aplicações em Investigação CriminalAula 9

4/4
Graph RAG é especialmente poderoso para análise de redes criminosas e fraudes.
  • Mapeamento de rede: identifica conexões entre suspeitos, empresas e transações
  • Análise temporal: 'Com quem X se comunicou nos 30 dias antes do crime?'
  • Detecção de padrão: identifica modus operandi comum entre diferentes casos
  • Cruzamento de informações: liga depoimentos, laudos e registros de forma automática
