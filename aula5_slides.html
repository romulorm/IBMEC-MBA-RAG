O Problema da Alucinação SilenciosaAula 5

1/6

Em sistemas jurídicos, respostas fluentes e plausíveis mas incorretas são mais perigosas que erros óbvios.

  • Alucinação simples: o LLM inventa um fato — fácil de detectar manualmente
  • Alucinação silenciosa: o LLM ignora o contexto recuperado e usa conhecimento do treinamento
  • Exemplo: recupera acórdão correto, mas cita prazo diferente do contexto
  • Solução: RAGAS mede a fidelidade ao contexto (Faithfulness) de forma automatizada
🎯

As 4 Métricas RAGASAula 5

2/6
RAGAS avalia simultaneamente o retriever e o gerador com 4 métricas complementares.
  • Faithfulness: % de claims na resposta que podem ser deduzidos dos contextos recuperados [0-1]
  • Answer Relevancy: quão bem a resposta aborda a pergunta feita [0-1]
  • Context Recall: % do ground truth coberto pelos contextos recuperados [0-1]
  • Context Precision: % dos contextos recuperados que são realmente relevantes [0-1]

Implementação RAGASAula 5

3/6

O dataset RAGAS requer três componentes: pergunta, contextos recuperados e resposta gerada.

  • Entrada: question + contexts + answer (+ ground_truth para Context Recall)
  • LLM-as-Judge: usa vLLM local para avaliar — sem custo de API externa
  • Resultado: DataFrame com scores por query + média agregada
  • Meta: Faithfulness > 0.8 e Context Precision > 0.7 para produção
Python 3.11+
from ragas import evaluate
from ragas.metrics import (
    faithfulness,
    answer_relevancy,
    context_recall,
    context_precision
)
from datasets import Dataset

# Dataset de avaliação
data = {
    "question": ["Quais são os requisitos da prisão preventiva?"],
    "answer": [resposta_gerada],
    "contexts": [[chunk1, chunk2, chunk3]],
    "ground_truth": ["Art. 312 CPP: fumus comissi delicti + periculum libertatis"]
}

dataset = Dataset.from_dict(data)
result = evaluate(dataset, metrics=[
    faithfulness, answer_relevancy,
    context_recall, context_precision
])
print(result.to_pandas())

DeepEval — Testes Unitários para LLMAula 5

4/6

DeepEval traz a cultura de testes unitários para pipelines de IA: assertions assertivas e CI/CD-friendly.

  • Testes como pytest: assert pipeline atende critério mínimo por métrica
  • 5 métricas: Faithfulness, Answer Relevancy, Contextual Precision, Recall, Hallucination
  • threshold: define o mínimo aceitável — falha se score < threshold
  • Integra com CI/CD: bloqueia deploy se qualidade degradou
Python 3.11+
import pytest
from deepeval import assert_test
from deepeval.metrics import FaithfulnessMetric, HallucinationMetric
from deepeval.test_case import LLMTestCase

@pytest.mark.parametrize("query,expected_context", test_cases)
def test_pipeline_faithfulness(query, expected_context):
    response = pipeline_rag(query)
    test_case = LLMTestCase(
        input=query,
        actual_output=response,
        retrieval_context=expected_context
    )
    metric = FaithfulnessMetric(threshold=0.8, model="llama3")
    assert_test(test_case, [metric])

LangFuse Scores APIAula 5

5/6

A Scores API do LangFuse permite enviar métricas RAGAS automaticamente e monitorar a qualidade em produção.

  • Score: par (traceId, name, value) — vinculado a uma execução específica
  • Histórico: acompanhe a evolução de Faithfulness ao longo do tempo
  • Alertas: configure thresholds para detecção de degradação
  • Dashboard: comparação visual Naive RAG vs Advanced RAG vs Hybrid
Python 3.11+
from langfuse import Langfuse

langfuse = Langfuse()

def avaliar_e_enviar(query: str, trace_id: str):
    # Rodar pipeline
    result = pipeline_rag(query)

    # Calcular RAGAS
    scores = ragas_evaluate([result])

    # Enviar para LangFuse
    langfuse.score(
        trace_id=trace_id,
        name="faithfulness",
        value=scores["faithfulness"],
        comment=f"Query: {query[:50]}"
    )
    langfuse.score(
        trace_id=trace_id,
        name="context_precision",
        value=scores["context_precision"]
    )

Cadência de Avaliação RecomendadaAula 5

6/6

Avaliação em múltiplas velocidades garante qualidade sem custo excessivo.

  • Online (100% das queries): Latência + Tokens — métricas de sistema, zero custo LLM
  • Sampling 10%: Faithfulness + Answer Relevancy via LangFuse Scores API
  • Diário: RAGAS no dataset de 50 queries padrão — detecta regressões
  • A cada deploy: DeepEval full suite — aprovação obrigatória antes de produção
